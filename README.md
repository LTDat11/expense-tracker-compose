# Expense Tracker App

An Android application for personal expense management, built with **Kotlin + Jetpack Compose**, following **MVVM architecture**, using **StateFlow** for state management, **Hilt (Dependency Injection)**, and **Room (SQLite).

---

## Features

- User login
- Add and delete expenses
- Offline data storage with Room
- Reactive UI using StateFlow
- Navigation with Navigation Compose

---

## Architecture

**MVVM + Repository Pattern**.

```text
UI (Jetpack Compose)
 ↓
ViewModel (StateFlow)
 ↓
Repository
 ↓
Room (SQLite)
```
---

## Tech Stack

- Kotlin
- Jetpack Compose
- StateFlow
- ViewModel
- Hilt (Dependency Injection)
- Room (SQLite)
- Navigation Compose
- Coroutines
- Material 3

---

## Getting Started

1. Clone the repository:
```bash
git clone git@github.com:LTDat11/expense-tracker-compose.git
```
2. Open the project with Android Studio
3. Run the app on an emulator or physical device
